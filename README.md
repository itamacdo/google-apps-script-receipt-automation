# Gerador Automático de Recibos – Google Apps Script

## Visão Geral
Este projeto automatiza a geração de recibos financeiros utilizando
o Google Sheets como fonte de dados e o Google Docs para criação dos documentos.

A solução foi desenvolvida em **Google Apps Script**, permitindo uma automação
100% integrada ao Google Workspace, sem necessidade de servidores externos.

---

## Funcionalidades
- Geração automática de recibos a partir de planilhas
- Criação de documentos no Google Docs
- Organização automática de arquivos no Google Drive
- Estrutura de pastas por período, entidade e quinzena
- Prevenção de geração de recibos duplicados
- Layouts configuráveis por entidade
- Código modular e reutilizável

---

## Tecnologias Utilizadas
- Google Apps Script
- JavaScript
- Google Sheets
- Google Docs
- Google Drive

---

## Caso de Uso
Projeto voltado para ambientes administrativos e financeiros que
necessitam gerar grande volume de documentos de forma padronizada,
segura e eficiente, reduzindo trabalho manual e riscos de erro.

Exemplos de aplicação:
- Setores financeiros
- Faturamento
- Controladoria
- Rotinas administrativas recorrentes

---

## Lógica da Solução
1. Leitura automática dos dados da planilha
2. Validação das informações obrigatórias
3. Identificação da entidade e do período
4. Definição automática da quinzena
5. Criação do recibo no Google Docs
6. Organização do arquivo no Google Drive
7. Verificação para evitar duplicidades

---

## Nota de Segurança
Todos os nomes, dados bancários, identificadores institucionais
e informações sensíveis foram **anonimizados ou substituídos**
nesta versão pública, mantendo apenas a lógica e a estrutura do projeto
para fins de demonstração em portfolio.
